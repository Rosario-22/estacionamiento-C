# Sistema de Estacionamiento en C
Proyecto de consola desarrollado en C que simula la gestión de un estacionamiento utilizando listas enlazadas dinámicas.

Funcionalidades
- Ingreso de autos
- Búsqueda por patente
- Retiro de autos
- Control de capacidad máxima

Conceptos aplicados
- Estructuras
- Punteros
- Listas enlazadas
- Memoria dinámica
- Modularización

Tecnologías
- Lenguaje C
- Compilado en Zinjai

Ejecución
Compilar y ejecutar "estacionamiento.c".

Mejoras futuras
- Persistencia en archivos
- Cálculo de tiempo de estadía
